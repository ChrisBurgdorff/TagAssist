Chrome Keyboard Shortcuts

  • Ctrl + S or [enter]: Click the Submit Button
  • Ctrl + B: Click the New Image Button
  • Ctrl + I: Set Payor Type to Insurance Carrier
  • Ctrl + Q: Set Cursor in top text box (Payor for Insurance Carrier Checks
  • Ctrl + E: Set Cursor in Endorsement Bank Name textbox (or Signature 1, this is the "jump" key)
  • Ctrl + G: Zoom In
  • Ctrl + H: Zoom Out
  • Ctrl + Y: Rotate
  • Ctrl + [number]: Zoom (When there are two images on a page)
  • 1: Zoom to Upper Left
  • 2: Move View to Right
  • 3: Zoom to Bottom Left
  • 4: Move View to Right
  • Ctrl + 5: Fill in Total Deposit Amount in Deposit Checks (pulled from last Deposit Slip)
  • Ctrl + [number]: Select Payor Type
  • 7: Personal
  • 8: Provider
  • 9: Other
  • Ctrl + [arrow]
  • Up: Paste last record entered
  • Down: Paste half of last record entered (back of check for Deposits, front of checks for Checks Out)
  • Left/Right: Cycle through template options
  • [backspace] default behavior prevented
